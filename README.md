# Sandhata Employee Onboarding Portal

A modern onboarding portal built using Next.js and Tailwind CSS.

## Features

* Employee Onboarding Guide
* Company Culture Hub
* Policy Repository
* Events & Engagement Calendar
* Contact Directory
* Mobile Responsive Navigation
* Quick Access Dashboard

## Tech Stack

* Next.js
* React
* Tailwind CSS
* Lucide React Icons
